# LocalClarity

LocalClarity is an AI-driven local search management platform for enterprises, agencies, and global brands. It provides tools and a REST API for managing Google Business Profiles, tracking local rankings, monitoring and responding to reviews across 50+ sources, and managing business listing data across Google, Apple, Bing, Facebook, and Waze.

## APIs

- **LocalClarity REST API** - Programmatic access to location data, reviews, rankings, and reporting. API keys are issued upon request via support@localclarity.com.
